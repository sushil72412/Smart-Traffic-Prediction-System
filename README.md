# 🚦 Smart Traffic Prediction System

An AI-powered Smart Traffic Prediction and Monitoring System using Machine Learning, Deep Learning, Weather APIs, Interactive Maps, and Real-Time Analytics.

## Features

* Traffic Prediction using Random Forest
* LSTM Deep Learning Forecasting
* Future Traffic Prediction
* Weather API Integration
* Interactive Traffic Congestion Map
* Dynamic Location Selection
* Streamlit Dashboard

## Technologies Used

* Python
* Streamlit
* Scikit-learn
* TensorFlow / Keras
* Folium
* OpenWeather API

## Future Enhancements

* Real-time CCTV monitoring
* YOLO vehicle detection
* Live traffic analytics
* Google Maps integration
* Smart congestion alerts

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
